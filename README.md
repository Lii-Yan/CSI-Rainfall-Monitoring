# CSI-Based Instantaneous Rainfall Monitoring (Sub-6 GHz)

This repository provides the open-source dataset and code for:

Yan Li et al.,  
"CSI-Based Instantaneous Rainfall Monitoring in Sub-6 GHz for ISAC Applications"

## Overview

This work presents:

- The first open-source multi-antenna CSI dataset under natural rainfall in sub-6 GHz
- Spatiotemporal channel analysis under real rainfall events
- CSI-Transformer RainNet for rainfall regression
